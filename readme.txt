Create professional WordPress Blog posts output with Elementor Blog page templates widget plugin by Web Devise. High-end layouts designs and Elementor controls in the box.

== Description ==
Build professional layouts for Blog Post out with WebDevise Elementor Blog page widgets addon for Elementor PageBuilder. 
The plugin includes widgets for outputting Blog posts with predesign widgets to have a nice view for Blog roll and posts output.
Also, widgets include Elementor controls to make easy your customization process for layouts.
Using this plugin you could easily combine different formats of output to make Blog posts output anywhere on your website.

= Mobile optimized =
Widgets posts output optimized for mobile usage to provide maximum involvement and easier navigation. Use controls and mobile view mode to make your own customization if you need it.

= SEO friendly =
All code in widgets done from SEO perspective to easier integration to any theme and your SEO strategy.


== Widgets ==
Many of these items you could easily control on the backend from the admin area.

= The Grid layout widget =
This view is an easy interaction format from UX point of view to attract users’ attention for your favorite posts or info you wanna underline. 
This layout consists of the base elements: post image, post title, author, date.
Nice look with sidebar and without sidebar area.

= List layout widget =
The best solution is when you need to have a long Blog roll. UX\UI optimized for best interaction with a big number of posts and easy navigation.
This layout consists of the base elements: post image, post title, author, date.


== Elementor Controls ==
Below is the list of controls that are included in the current version. Don’t forget to make updates to have up to date features:

- **Post category selection**.  You could select custom category posts for the output
- **Number of posts**. Set up a number of posts for output per page
- **Pagination**. Switch ON/OFF pagination on the page depends on your needs
- **Author**. Control for showing author name and gravatar
- **Typography**. Control to manage typography for Title
- **Color**. Elementor control for coloring texts

The current version of the Product is workable with Elementor free and Elementor PRO

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages > Add New
4. Press the 'Edit with Elementor' button.
5. Now you can drag and drop widgets (WebDevise section: Post Grid, Post List) from the left panel onto the content area, as well as add new sections and columns that make up the page structure.
